# 프롬프트 4-1: 수요 구조 분석

## 목적
이 기업의 제품/서비스에 대한 수요 구조를 분석하여, 향후 3-5년간 지속 성장 가능성을 평가한다.

## 분석 지시

아래 제공된 기업 데이터를 바탕으로 다음 항목을 분석하세요:

### 1. 핵심 고객 구조 (TAM × 집중도)
- 상위 3개 고객이 매출에서 차지하는 비중
- 고객 유형: 빅테크 / 방산·정부 / 일반 기업 / 소비자
- 단일 고객 의존도 리스크 (최대 고객 30% 초과 여부)

### 2. 수요 드라이버 지속성
- 구조적 드라이버 (AI CAPEX, 에너지 전환, 리쇼어링 등) vs 일시적 수요
- 수주잔고/파이프라인 성장 추세
- 고객사의 CAPEX 계획이 이 기업의 매출 성장을 뒷받침하는지

### 3. TAM 확장 가능성
- 현재 침투율 vs 잠재 시장
- 신규 지역·산업으로의 확장 경로
- 경쟁사 대비 시장점유율 변화 추이

### 4. 수요 탄력성 & 대체 리스크
- 제품 대체 가능성 (기술 변화, 경쟁 신제품)
- 가격 인상 시 수요 감소 가능성
- 고객 전환 비용 (높을수록 방어적)

## 출력 형식 (JSON)
```json
{
  "demand_score": 8,
  "demand_summary": "AI CAPEX 직접 수혜, 빅테크 3사 고객 집중, 수주잔고 2.3배 YoY",
  "key_demand_drivers": ["NVIDIA 메모리 수요", "CSP CAPEX 증설", "온디바이스 AI"],
  "concentration_risk": "중간 (최대 고객 32%)",
  "tam_growth_rate_pct": 35,
  "demand_durability": "구조적",
  "risks": ["단일 고객 의존", "기술 전환 리스크"]
}
```

demand_score: 1-10점 (10점 = 구조적 수요 폭발 + 대체 불가 + TAM 급성장)
