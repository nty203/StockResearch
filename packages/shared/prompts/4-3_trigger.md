# 프롬프트 4-3: 트리거 이벤트 분석

## 목적
향후 12-24개월 내 주가 10배 상승을 촉발할 수 있는 이벤트를 찾아, 그 타이밍과 크기를 평가한다.

## 분석 지시

아래 제공된 공시·뉴스·재무 데이터를 바탕으로 다음을 분석하세요:

### 1. 현재 진행 중인 트리거
아래 15가지 트리거 유형 중 현재 데이터에서 확인되는 것을 모두 식별하세요:
1. 빅테크 단일 대형 수주
2. 신규 생산 설비 착공 / CAPEX 사이클 개시
3. 글로벌 메가 계약 (방산·원전·인프라)
4. 빅테크 전략적 파트너십 / 독점 공급 계약
5. 시장 독점 확보 / 경쟁사 퇴출
6. 핵심 기술 특허 / 세계 최초 양산
7. 영업이익률 급등 (구조적 전환점)
8. 내부자 대규모 매수
9. 기관·외국인 집중 매수
10. 규제 장벽 해소 / 인허가 획득
11. 공급 병목 수혜 (유일 공급사 포지션)
12. 핵심 원자재 가격 하락 (원가 급개선)
13. 지정학 수혜 (리쇼어링, CHIPS Act 등)
14. 사업 분할 / 스핀오프 가치 재평가
15. 어닝 서프라이즈 + 가이던스 상향

### 2. 트리거 크기 평가
- 각 트리거의 매출 임팩트 추정 (현재 매출 대비 %)
- 주가 멀티플 재평가 가능성

### 3. 카탈리스트 타임라인
- 향후 6개월 내 / 12개월 내 / 24개월 내 예상 이벤트
- 주요 일정 (실적 발표, 공장 완공, 규제 결정 등)

## 출력 형식 (JSON)
```json
{
  "trigger_score": 9,
  "active_triggers": ["빅테크_단일수주", "CAPEX_증설"],
  "trigger_timeline": {
    "6m": "2공장 착공 완료",
    "12m": "NVIDIA 2차 수주 발표 예상",
    "24m": "양산 풀가동 → 매출 2배"
  },
  "revenue_impact_pct": 85,
  "catalyst_summary": "TC본더 독점 수혜 + HBM4 사이클 개시"
}
```

trigger_score: 1-10점 (10점 = 복수의 강력한 트리거가 동시에 12개월 내 발화)
